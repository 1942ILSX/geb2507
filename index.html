ABOUT THE GAME

Why Script Spire?

Modern society relies heavily on computers, yet many people have little understanding of how programs actually operate. For beginners, programming often feels abstract and intimidating, with steep logical requirements that discourage sustained learning. Script Spire is designed as a serious game to address this gap by turning programming concepts into an interactive, visual, and low-pressure experience.

Rather than teaching a specific language, the game focuses on core computational thinking: step-by-step reasoning, planning, and debugging. By watching a robot follow their written instructions, players see how computers execute commands line by line, building a concrete mental model of program flow.

HOW IT WORKS

Core Gameplay & Learning Loop

Script Spire is a 2D puzzle platformer where the only way to move the robot is by writing a small script in a custom mini-language and pressing Run. Each level presents a layout of platforms, spikes, moving elements, and a goal object. Players must plan a sequence of commands that will safely guide the robot to the target.

  • right / left / stop – continuous horizontal movement
  • jump – jumping only when on the ground
  • wait(x) – wait for x seconds
  • wait_ground / wait_air – wait until a specific state is met

These simple commands are powerful enough to express rich behaviors such as chained jumps on moving platforms or precisely timed runs across hazards, while remaining understandable to complete beginners.

The gameplay loop mirrors real programming:

  • Inspect the level and goal
  • Write a script to control the robot
  • Run the code and watch what happens
  • Debug and refine the instructions
  • Repeat until the solution works

Line highlighting during execution shows which command is currently running, helping players trace the link between code and behavior. A reset button lets them retry quickly without losing their script, encouraging experimentation rather than fear of failure.

DESIGN HIGHLIGHTS

Serious Game Features

Concept-first Design

The game focuses on universal programming concepts like sequencing, state-based actions, and timing, rather than any specific language syntax.

Puzzle-based Progression

Levels gradually introduce new elements—hazards, moving platforms, and timing challenges—so players deepen their reasoning instead of relying on reflexes.

Debugging Made Visual

Robot movement, line highlighting, and easy resets help players see where their logic breaks and how small changes affect the outcome.

Experimentation-Friendly

Failure is cheap: scripts persist across runs, and players are encouraged to “try, observe, adjust” — the core mindset of real-world programming.

EVALUATION

What Did Players Think?

Script Spire was evaluated with 44 participants using a mixed-methods approach combining in-game analytics, a post-game questionnaire, and follow-up interviews. Overall, players reported positive learning experiences and engagement with the game’s programming logic.

Overall Average Rating
3.67 / 5
on an 11-item Likert scale
Learning Effectiveness
3.75 / 5
understanding & confidence
Motivational Impact
3.76 / 5
interest in learning more

Qualitative feedback suggests that the game helped demystify basic programming, especially the idea that computers follow instructions step by step. Participants appreciated the clear visual feedback and low-pressure environment, while also suggesting future enhancements such as more advanced mechanics and additional tutorials.
